# Category Theory

Interactive category theory learning platform with diagrams and exercises.

## Overview

A web-based educational tool for exploring category theory concepts through interactive diagrams and hands-on exercises. Built with TypeScript and Vite.

## Getting Started

```bash
npm install
npm run dev
```

Open `index.html` or visit the local dev server to begin exploring category theory interactively.

## Tech Stack

- TypeScript
- Vite
- HTML/CSS
